# 爬虫工具
    - HTTrack Website Copier
    - Goodle:
        - site: 主机名 关键词 比如 SSN(社会保障号码), credit card 信用卡等
        - filetype: 文件类型
        - allintitle: 标题中的关键词
        - inurl: url 中的关键词
        - related:  找相似的网页
        - link: 该网站连接的其他网站

        - Johnny Long 约翰尼-朗 的 Google Hacking Database GHDB, Goodle黑客攻击数据库 [ http://johnny.ihackstuff.com/ghdb ], 著有 Google Hacking for Penetration Testers

# 攻击
    - 在未做检验的表单中提交大量数据
    - URL 篡改, 利用服务器url解析漏洞
    - 修改请求字段
    - 代码注入, SQL注入
    - XSS跨站脚本, 如伪造登录页
    - Web 攻击工具:
        - OWASP WebGoat Project [ www.owasp.org/index.php/Category:OWASP_WebGoat_Project ]
        - Hacme Tools           [ www.foundstone.com/us/resources-free-tools.asp ]
